# 修改理由和意见

1. 原文档包含两个明确区分的部分：
   - 开篇说明属于解释（Explanations）范畴，用于阐述文档目的和适用场景
   - 虚拟机列表属于参考（Reference）范畴，提供静态配置信息

2. 结构优化建议：
   - 拆分说明文字和配置表格为独立模块
   - 为两个部分建立双向锚点链接
   - 规范标题格式（删除多余空格）
   - 保持原有声明和表格内容完整
